# LAB 6 : Contexte et État d'un Agent

Ce laboratoire présente l'utilisation du **contexte** et de l'**état** dans les agents LangChain.

## Contenu du TP

* Définition d'une classe de contexte `ColourContext`
* Création d'un agent sans contexte
* Création d'un agent avec contexte
* Modification dynamique du contexte
* Définition d'un état personnalisé avec `AgentState`
* Mise à jour de l'état avec `InMemorySaver`
* Lecture des informations enregistrées dans l'état

## Technologies utilisées

* LangChain
* LangGraph
* Ollama (`llama3.2:3b`)
* Python
